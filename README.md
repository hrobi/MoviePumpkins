## setup
Töltsd le a `java 21 coretto` openJDK toolkit-et a backend-hez.

A frontend-hez szükséged lesz nodejs environment-re, javaslom az nvm program használatát,
mellyel könnyen lehet node verziók közül választani. A projekthez használt node verzió az `api`,
illetve `frontend` könyvtárak `.nvmrc` file-jában található.
